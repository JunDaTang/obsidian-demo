## 目录职责

### 00_System
仓库运行说明、规则、流程、标签说明。

### 01_Projects
PARA 的 Projects：有明确目标、有结束时间的内容项目。

### 02_Areas
PARA 的 Areas：长期经营的主题，例如 AI内容工厂、个人品牌、内容方法论。

### 03_Resources
PARA 的 Resources：外部资料、可复用素材、案例、数据、标题库、CTA 库等。

### 04_Knowledge
Zettelkasten 知识层：Fleeting / Literature / Permanent / MOCs。

### 05_Content
内容流水线：Inbox / Topic / Angle / Draft / Published / Feedback / Repurposing。

### 06_Archives
过期项目、旧内容、旧知识与失效观点。

### 90_Templates
所有模板统一收纳。

### 99_Dashboard
给人和 AI 的导航页、索引页、活跃工作台。
